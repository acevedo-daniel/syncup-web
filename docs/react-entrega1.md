# Proyecto - Entrega 1

Fecha de entrega: 01/11 - 23:59 horas
Estado: Obligatoria para aprobar el curso

## Objetivo

Construir la estructura base de una aplicación en React aplicando los fundamentos vistos hasta el momento.

Cada alumno debe definir la temática de su aplicación y adaptar los requerimientos al contexto elegido.

## Temas a Aplicar

- Vite
- JSX
- Componentes
- Estilos (CSS Modules o alternativa elegida)
- Renderizado de listas
- Uso de children para contenedores reutilizables
- Despliegue

## Requerimientos

### 1. Proyecto configurado con Vite + React

Debe compilar y correr correctamente.

### 2. Estructura básica de componentes

- Navbar o encabezado
- Lista principal de elementos (items, cards, publicaciones, productos, etc.)

### 3. Renderizado de datos mockeados

Debe existir una constante con datos falsos.

### 4. Estilos

Debe haber coherencia visual (no se exige diseño complejo).

### 5. Contenedores con children

Debe existir al menos un componente genérico reutilizable que reciba children.

### 6. Renderizar los items

Cada card debe mostrar información básica del elemento.

### 7. (Opcional) Detalle por item

Se puede mostrar el detalle mediante renderizado condicional.

## Forma de Entrega

1. Subir el proyecto a un repositorio remoto (GitHub)
2. Desplegar la aplicación (Vercel recomendado)
3. Enviar:
   - Link del repositorio
   - Link de la aplicación desplegada
